⚠️  IMPORTANT NOTES:

1. MONGO_URI Format (Critical):
   ✓ mongodb+srv://username:password@cluster.mongodb.net/spin?retryWrites=true&w=majority
   ✗ mongodb://localhost:27017  (local - won't work on Render)
   ✗ mongodb://user:pass@host   (old format - not recommended)

2. Special Characters in Password:
   - If password contains @, #, %, or other special chars
   - URL-encode them: 
     @ = %40
     : = %3A
     # = %23
   - Example: password@123 → mongodb+srv://user:password%40123@cluster...

3. Whitelist IP in MongoDB Atlas:
   - Development: Add your local IP
   - Production (Render): Add Render's IPs (0.0.0.0/0 for testing)

4. Admin Authentication:
   - All POST/PUT/DELETE routes require x-admin-key header
   - Header value must match ADMIN_PASSWORD environment variable
   - GET routes don't require authentication

5. Logs to Monitor:
   - "✅ MongoDB Connected Successfully" = Success
   - "❌ MongoDB Connection Failed" = Check MONGO_URI
   - "[timestamp] POST /api/units" = Request received
   - "console.error" = Problem occurred, check error details
